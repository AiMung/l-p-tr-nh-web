⚡️ Fun fact Sự ra đời của CSS:

CSS (Cascading Style Sheets), một công nghệ quan trọng trong lập trình web, được phát triển bởi Hakon Wium Lie vào năm 1994. Tuy nhiên, khi CSS lần đầu tiên được giới thiệu, nhiều người không mấy tin tưởng vào tính hữu ích của nó.

Thực tế, vào thời điểm đó, phần lớn các nhà phát triển web đều sử dụng HTML để định dạng và thiết kế trang web, và họ cảm thấy CSS có thể không cần thiết. Nhưng chỉ trong vài năm, CSS đã trở thành một phần không thể thiếu trong phát triển web, cho phép các nhà thiết kế dễ dàng thay đổi giao diện của trang web mà không cần phải thay đổi cấu trúc HTML.

Ngày nay, CSS không chỉ giúp định dạng văn bản và bố trí trang web, mà còn hỗ trợ các kỹ thuật nâng cao như animations và responsive design, giúp các trang web hoạt động tốt trên mọi thiết bị.
